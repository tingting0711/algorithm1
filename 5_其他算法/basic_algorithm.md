> 约数之和

- 通用：求**等比数列之和**
- $ N = p_1^{a_1} * p_2^{a_2} * …… * p_k^{a_k}$
- N有 $ f[N] = (a_1 + 1) * (a_2 + 1) * ...... * (a_k + 1) $个约数
- 约数之和为 $ (p_1^0 + p_1^1 + … + p_1^{a_1}) * … *  (p_k^0 + pk^1< + … + p_k^{a_k})$
- N^B的约数之和  $ (p_1^0 + p_1^{1B} + … + p_1^{a_1B}) * … *  (p_k^0 + pk^{1B}< + … + p_k^{a_kB})$
- 对于单个式子$(p_1^0 + p_1^{1B} + … + p_1^{a_1B})$ 
  - $ sum(p, k) = p^0 + p^1 + ... + p^{k - 1} = (p^0 + p^1 + ... + p^{k/2 - 1}) + (p^{k/2} + p^{k/2 + 1} + ... + p^{k - 1})$
  - $sum(p, k) = (1 + p^{k / 2})*(p^0 + p^1 + ... + p^{k/2 - 1}) = (1 + p^{k / 2}) * sum(p, k / 2)$ ，其中 $p^{k / 2}$使用快速幂.
  - k为奇数时，$sum(p, k) = sum(p, k - 1) + p^k$
