> 埃氏筛法

```
//朴素版，时间复杂度 nlogn
f(n) = 1 + 1/2 + 1/3 + ... + 1/n = ln n + c 
for(int i = 2; i <= n; i++)
{
		if(!st[i])
		{
				primes[cnt ++] = i;
		}
		else break;
		for(int j = 2i; j <= n; j += i)
				st[j] = true;
}

//优化版，时间复杂度 nloglogn
1/2 + 1/3 + 1/5 + 1/7 + ....(质数)... = loglogn
for(int i = 2; i <= n; i++)
{
		if(!st[i])
		{
				primes[cnt ++] = i;
				for(int j = 2i; j <= n; j += i)
				st[j] = true;
		}
		else break;
}
```



> 线性筛法

```
时间复杂度 O(n)
for(int i = 2; i <= n; i++)
{
		if(!st[i])
		{
				primes[cnt ++] = i;
		}
		for(int j = 0; primes[j] * i <= n; j ++)
		{
				st[primes[j] * i] = true;
				if(i % primes[j])break;
				//每个数只会被最小质因子删除一次
		}
}
```

## 约数

- $ N = p_1^{a_1} * p_2^{a_2} * …… * p_k^{a_k}$

- N有 $ f[N] = (a_1 + 1) * (a_2 + 1) * ...... * (a_k + 1) $个约数

- 约数之和为 $ (p_1^0 + p_1^1 + … + p_1^{a_1}) * … *  (p_k^0 + pk^1< + … + p_k^{a_k})$

- 1~N 中，所有约数之和 $ \sum_{i = 0}^N f[i] = \frac{N}{1} +  \frac{N}{2} + ... +  \frac{N}{N} = N \log N $

  - 对于$ 0 \leq N \leq 2*(10)^9 $, 约数个数 $ f[N] \approx 1600 $
  
- 最大公约数与最小公倍数关系：
  
  - (x, y) * [x, y] = x * y
